## 🛠️ Cara Upload
1. Hubungkan OLED ke ESP32:
    - VCC ➔ 3.3V
    - GND ➔ GND
    - SDA ➔ GPIO21
    - SCL ➔ GPIO22
2. Upload sketch menggunakan Arduino IDE
3. Pastikan library:
    - Adafruit_GFX
    - Adafruit_SSD1306
4. OLED akan menampilkan gambar emoticon 😚
